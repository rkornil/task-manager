# Менеджер завдань (Task Manager)

Це мій навчальний проект на Java. Я тільки почав вивчати Java, тому вирішив
написати щось просте, але зі своєю логікою — консольний менеджер завдань 🙂

## Що вміє програма

- Додавати нові завдання (назва, опис, пріоритет)
- Показувати список усіх завдань
- Позначати завдання як виконане
- Видаляти завдання зі списку
- Показувати статистику (скільки завдань виконано, а скільки ще ні)

## Як запустити

1. Склонуй репозиторій
2. Відкрий папку проекту у своїй IDE (я використовував IntelliJ IDEA)
3. Запусти файл `Main.java`

Або через термінал:

\`\`\`bash
cd src
javac Main.java Task.java TaskManager.java
java Main
\`\`\`

## З чого складається проект

- `Task.java` — клас, який описує одне завдання
- `TaskManager.java` — клас, який керує списком завдань (додає, видаляє, рахує статистику)
- `Main.java` — головний клас із меню та взаємодією з користувачем

## Що я тут використав

- Класи та об'єкти (ООП)
- ArrayList для зберігання списку завдань
- Scanner для введення даних з клавіатури
- Цикли `while` та `for`
- Конструкцію `switch-case`
- `try-catch` для обробки некоректного вводу

## Плани на майбутнє

- Додати збереження завдань у файл, щоб вони не зникали після закриття програми
- Додати редагування вже створених завдань
- Спробувати зробити графічний інтерфейс (GUI)
